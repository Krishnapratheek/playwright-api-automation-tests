# Playwright API Automation Tests

This project contains automated API tests for the JSONPlaceholder REST API using Playwright and TypeScript.

## 📌 API Endpoint
https://jsonplaceholder.typicode.com/posts

---

## 🚀 Test Coverage

### Core API Tests
- GET all posts
- GET single post
- POST create post
- PUT update post
- DELETE post

### Error Handling
- Non-existent post (404)
- Invalid request data

### Edge Cases
- Query parameters
- Response headers validation
- Response time validation
- Data type validation

---

## 🛠 Tech Stack

- Playwright
- TypeScript
- Node.js
